5. Features Table with Header Groups

HTML Table Features and Attributes
HTML Table Essential Reference
Category Attribute/Element Description Example
Basic Table Structure
Elements <table> Defines an HTML table <table>...</table>
<tr> Defines a table row <tr>...</tr>
<th> Defines a header cell <th>Header</th>
<td> Defines a standard data cell <td>Data</td>
<caption> Defines a table caption <caption>Title</caption>
Table Sections
Section Elements <thead> Groups the header content <thead>...</thead>
<tbody> Groups the body content <tbody>...</tbody>
<tfoot> Groups the footer content <tfoot>...</tfoot>
Cell Spanning
Spanning Attributes colspan Number of columns a cell spans <td colspan="3">...</td>
rowspan Number of rows a cell spans <td rowspan="2">...</td>
Column Groups
Column Elements <colgroup> Specifies a group of columns <colgroup>...</colgroup>
<col> Specifies column properties <col style="width: 20%">
Accessibility Attributes
Accessibility scope Associates header cells with data cells <th scope="col">...</th>
headers Associates data cells with header cells <td headers="id1 id2">...</td>
id Unique identifier for a cell <th id="name">Name</th>

Key HTML Table Attributes and Elements Demonstrated

  • Basic Structure: table, tr, th, td elements
  • Table Sections: thead, tbody, tfoot
  • Cell Spanning: colspan, rowspan attributes
  • Table Caption: caption element
  • Column Groups: colgroup, col elements
  • CSS Properties: border-collapse, border-spacing, nth-child selectors
  • Responsive Design: Media queries for small screens
  • Accessibility Features: data-label attributes for responsive tables
  • Nested Tables: Tables within table cells
